API Reference
All endpoints for the Inklude API. Authentication required for identity and analysis endpoints. Set your API key in the header above.
Public Endpoints
GET /api/v1/health
Basic liveness probe. Returns API status.
GET /api/v1/health/ready
Readiness probe. Checks if NLP model is loaded.
GET /api/v1/neo-pronouns/
List all neo-pronoun sets. Optional query: ?popularity=common|moderate|emerging|historical
GET /api/v1/neo-pronouns/check?token=ze
Check if a word is a recognized neo-pronoun form.
GET /api/v1/neo-pronouns/{label}
Get a specific set by label (e.g. ze/hir).
Authenticated Endpoints
Require X-API-Key header.
POST /api/v1/analyze/text
Analyze text for gendered language and pronoun issues.
{"text": "...", "tone": "gentle|direct|research_backed"}
POST /api/v1/analyze/batch
Analyze multiple texts in one request.
{"texts": ["...", "..."], "tone": "gentle"}
POST /api/v1/analyze/check-pronouns
Check pronoun usage against registered identities.
{"text": "...", "person_ids": ["uuid", ...], "tone": "gentle"}
CRUD /api/v1/identities/
Full CRUD for identity profiles. Supports pronoun sets (including neo-pronouns) and preferences.
POST / | GET / | GET /{id} | PUT /{id} | DELETE /{id}
GET /{id}/pronouns | PUT /{id}/pronouns | PUT /{id}/preferences
